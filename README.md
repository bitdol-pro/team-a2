# OneDay
BoostCamp 2019 iOS A2

넘사벽 저널링 서비스로 알려져 있는 **[DayOne](https://dayone.me/)** 모방 프로젝트

## 핵심 기능
- 일기 작성
- 일기 검색 및 필터
- 다양한 모드로 일기 보여주기
  - 타임라인
  - 캘린더
  - 지도
  - 사진
- 저널 단위로 일기 관리
- iCloud 동기화
